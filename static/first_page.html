In-door Location Tracker

Our invention originated from the wish to track the hidden carts inside a building, since many people don't feel obligated to return them after usage. We aim to design and implement an indoor object tracking device -- Cart-E.

Cart-E is specifically designed to track indoor positions. Instead of relying on GPS signals, which become ambiguous in-door attenuated by roofs and walls, Cart-E uses open-source information such as WIFI-router's public IP addresses to ensure accuracy and avoid privacy issues.

With our in-door tracking device Cart-E, which is small, inexpensive, and energy-efficient, we can locate desired objects inside a building simply through accessing a website. This technology produces an affordable device with little maintenance requirement that provides detailed information about the object location in 3D space. This allows the user to be able to retrieve the object fast and easy.

by Alga Peng, Xiangyi Zhao
